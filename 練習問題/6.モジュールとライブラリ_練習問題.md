# 練習問題：モジュールとライブラリ

このファイルでは、「6.モジュールとライブラリ.md」で学んだモジュールの使い方について、いくつかの問題に挑戦してみましょう。

---

### 問題1：ランダムな数字

Pythonの標準ライブラリである `random` モジュールは、乱数（ランダムな数）を生成するための機能を提供します。

1.  `random` モジュールをインポートしてください。
2.  `random.randint(a, b)` 関数を使うと、`a` から `b` までの範囲に含まれるランダムな整数を1つ生成できます。この関数を使って、1から6までのランダムな整数（サイコロの目のイメージ）を生成し、表示してください。
3.  プログラムを何回か実行して、毎回違う数字が表示されることを確認してください。

---

### 問題2：今日の日付

`datetime` モジュールは、日付や時刻を扱うための機能を提供します。

1.  `datetime` モジュールから、日付を扱う `date` クラスだけを `from ... import ...` を使ってインポートしてください。
2.  `date.today()` を使うと、今日の日付を取得できます。これを実行して、結果を変数 `today` に代入してください。
3.  `today` を `print()` で表示してください。
4.  `today.year`, `today.month`, `today.day` のように、年・月・日をそれぞれ個別に取り出すことができます。これらを使って、「今日は○○年△△月××日です。」という形式で表示してください。

---

### 問題3：円の面積（mathモジュール）

`math` モジュールは、数学的な計算のための定数や関数を提供します。

1.  `math` モジュールを `m` という別名（エイリアス）でインポートしてください。
2.  円の面積は「半径 × 半径 × 円周率」で求められます。`math` モジュールに含まれる円周率 `m.pi` と、べき乗を計算する `m.pow(x, y)` （xのy乗）を使って、半径が `5` の円の面積を計算し、表示してください。

### 問題1
```python
# 1. モジュールをインポート
import random

# 2. 1から6までのランダムな整数を生成して表示
dice_roll = random.randint(1, 6)
print(f"サイコロの目は {dice_roll} でした。")
```

### 問題2
```python
# 1. datetimeモジュールからdateクラスをインポート
from datetime import date

# 2. 今日の日付を取得
today = date.today()

# 3. そのまま表示
print(today)

# 4. フォーマットして表示
print(f"今日は{today.year}年{today.month}月{today.day}日です。")
```

### 問題3
```python
# 1. mathモジュールをmとしてインポート
import math as m

radius = 5

# 2. 円の面積を計算
# m.pow(radius, 2) は radius ** 2 と同じ意味
area = m.pow(radius, 2) * m.pi

print(f"半径{radius}の円の面積は {area} です。")
```
