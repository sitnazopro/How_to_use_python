# 練習問題：変数、データ型、演算子

このファイルでは、「1.変数とデータ型と演算子.md」で学んだ内容を使って、いくつかの問題に挑戦してみましょう。

---

### 問題1：自己紹介

1.  `name` という変数にあなたの名前（文字列）を代入してください。
2.  `age` という変数にあなたの年齢（整数）を代入してください。
3.  `print()` 関数を使って、コンソールに「私の名前は○○です。年齢は△△歳です。」と表示させてください。○○と△△には、変数の中身が入るようにします。

**ヒント:** f-string を使うと、文字列の中に変数を埋め込むのが簡単です。

---

### 問題2：長方形の面積と周の長さ

1.  `width` という変数に `15` を、`height` という変数に `8` を代入してください。
2.  この2つの変数を使って、長方形の面積 (`area`) と周の長さ (`perimeter`) を計算し、それぞれの結果を新しい変数に代入してください。
    -   面積 = 幅 × 高さ
    -   周の長さ = (幅 + 高さ) × 2
3.  計算結果を「面積: ○○」「周の長さ: △△」のように表示してください。

---

### 問題3：比較と論理

1.  `score` という変数に `85` を代入してください。
2.  `score` が `80` 以上であるかどうかを比較演算子を使って判定し、結果（`True` または `False`）を表示してください。
3.  `score` が `60` 以上 **かつ** `90` 未満であるかどうかを論理演算子を使って判定し、結果を表示してください。


---

### 問題1
```python
name = "山田太郎"  # あなたの名前に書き換えてください
age = 30         # あなたの年齢に書き換えてください

print(f"私の名前は{name}です。年齢は{age}歳です。")
```

### 問題2
```python
width = 15
height = 8

area = width * height
perimeter = (width + height) * 2

print(f"面積: {area}")
print(f"周の長さ: {perimeter}")
```

### 問題3
```python
score = 85

# 80点以上か？
is_over_80 = score >= 80
print(f"80点以上か: {is_over_80}")

# 60点以上かつ90点未満か？
is_in_range = score >= 60 and score < 90
print(f"60点以上90点未満か: {is_in_range}")
```
