# 練習問題：条件分岐 (if文)

このファイルでは、「3.条件分岐.md」で学んだ `if`, `elif`, `else` を使って、いくつかの問題に挑戦してみましょう。

---

### 問題1：飲み物の提案

1.  `temperature` という変数に、現在の気温（整数）として `28` を代入してください。
2.  `if` 文を使い、`temperature` が `25` 度より大きい場合に「冷たい飲み物がおすすめです。」と表示するプログラムを書いてください。
3.  `temperature` の値を `20` に変えてみて、何も表示されなくなることを確認してください。

---

### 問題2：偶数か奇数か

1.  `number` という変数に、好きな整数を代入してください。
2.  `if-else` 文を使って、その `number` が偶数か奇数かを判定するプログラムを書いてください。
    -   偶数なら「○○は偶数です。」と表示します。
    -   奇数なら「○○は奇数です。」と表示します。
3.  ヒント：ある数を `2` で割った余りが `0` であれば、その数は偶数です。余りを計算するには `%` 演算子が使えます。

---

### 問題3：映画の料金

年齢に応じて映画の料金を判定するプログラムを `if-elif-else` 文を使って書いてください。

1.  `age` という変数に、年齢を表す整数を代入してください。
2.  以下の条件に従って、料金を表示してください。
    -   12歳以下の場合：「料金は500円です。」
    -   13歳から18歳の場合：「料金は1000円です。」
    -   それ以外（19歳以上）の場合：「料金は1800円です。」
3.  `age` の値に `10`, `15`, `25` などを代入して、表示が正しく変わるか確認してください。


---

### 問題1
```python
temperature = 28

if temperature > 25:
    print("冷たい飲み物がおすすめです。")

# temperature = 20 に変えると、ifの条件がFalseになるため何も出力されない
```

### 問題2
```python
number = 17  # 好きな整数に変えて試してください

if number % 2 == 0:
    print(f"{number}は偶数です。")
else:
    print(f"{number}は奇数です。")
```

### 問題3
```python
age = 15  # 色々な年齢で試してください

if age <= 12:
    print("料金は500円です。")
elif age <= 18:  # 13歳から18歳 (age > 12 は最初のifでチェック済み)
    print("料金は1000円です。")
else: # 19歳以上
    print("料金は1800円です。")
```
