# 練習問題：基本的なデータ構造

このファイルでは、「2.基本的なデータ構造.md」で学んだリスト、タプル、辞書、セットを使って、いくつかの問題に挑戦してみましょう。

---

### 問題1：好きなものリスト

1.  あなたの好きな食べ物を3つ以上入れたリスト `favorite_foods` を作成してください。
2.  リストの2番目の要素（インデックスは `1`）を `print()` で表示してください。
3.  `append()` メソッドを使って、リストの末尾に新しい食べ物を1つ追加してください。
4.  最後に、更新されたリスト全体を表示してください。

---

### 問題2：個人のプロフィール

1.  個人のプロフィール情報を格納するための辞書 `profile` を作成してください。
2.  辞書には、`"name"`（名前）、`"age"`（年齢）、`"city"`（住んでいる都市）の3つのキーを含め、それぞれのキーに対応する値（あなたの情報）を設定してください。
3.  辞書から `"age"` の値を取得して表示してください。
4.  辞書に `"job"`（職業）という新しいキーと値のペアを追加してください。
5.  最後に、更新された辞書全体を表示してください。

---

### 問題3：重複しない色のリスト

1.  以下の様に、色が重複して含まれているリスト `colors` があります。
    ```python
    colors = ["red", "blue", "green", "red", "yellow", "blue"]
    ```
2.  セット (`set`) の性質を利用して、このリストから重複する色を取り除いてください。
3.  重複がなくなった後の、ユニークな色の集まりを表示してください。（ヒント：`set()` 関数が使えます）


---

### 問題1
```python
# 1. リストを作成
favorite_foods = ["ラーメン", "寿司", "カレー"]

# 2. 2番目の要素を表示
print(f"2番目に好きなもの: {favorite_foods[1]}")

# 3. 新しい食べ物を追加
favorite_foods.append("焼肉")

# 4. 更新されたリストを表示
print(f"新しいリスト: {favorite_foods}")
```

### 問題2
```python
# 1 & 2. 辞書を作成
profile = {
    "name": "鈴木一郎",
    "age": 28,
    "city": "大阪"
}

# 3. 年齢を表示
print(f"年齢: {profile['age']}")

# 4. 職業を追加
profile["job"] = "プログラマー"

# 5. 更新された辞書を表示
print(f"新しいプロフィール: {profile}")
```

### 問題3
```python
colors = ["red", "blue", "green", "red", "yellow", "blue"]

# セットに変換して重複を削除
unique_colors = set(colors)

# 結果を表示
print(f"重複しない色のリスト: {unique_colors}")
```
