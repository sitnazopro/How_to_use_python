# 練習問題：繰り返し処理 (ループ)

このファイルでは、「4.繰り返し処理.md」で学んだ `for` ループと `while` ループを使って、いくつかの問題に挑戦してみましょう。

---

### 問題1：リストの要素をすべて表示

1.  `p_languages` というリストに、`["Python", "Java", "JavaScript", "C++"]` を代入してください。
2.  `for` ループを使って、リストの各要素を一つずつ取り出し、「プログラミング言語: ○○」という形式で表示してください。

---

### 問題2：1から10までの合計

1.  `total` という変数に `0` を代入して、合計値を保存する準備をします。
2.  `for` ループと `range()` 関数を使って、1から10までの数値を順番に `total` に足し合わせてください。
3.  ループが終わった後、「1から10までの合計は○○です。」と最終的な合計値を表示してください。

---

### 問題3：3の倍数だけを表示

1から20までの数の中で、3の倍数だけを表示するプログラムを書いてください。

-   `for` ループと `range(1, 21)` を使って、1から20までの数を調べます。
-   ループの中で `if` 文を使い、その数が3の倍数かどうかを判定します。（ヒント：`%` 演算子を使います）
-   3の倍数であれば、その数を `print()` で表示します。

**挑戦問題:** `continue` を使って、3の倍数**でない**場合に処理をスキップする方法でも書いてみましょう。

### 問題1
```python
p_languages = ["Python", "Java", "JavaScript", "C++"]

for lang in p_languages:
    print(f"プログラミング言語: {lang}")
```

### 問題2
```python
total = 0

# range(1, 11) で 1 から 10 までの数値を生成
for i in range(1, 11):
    total = total + i  # total += i とも書ける

print(f"1から10までの合計は{total}です。")
```

### 問題3

**通常版**
```python
for i in range(1, 21):
    # 3で割った余りが0なら3の倍数
    if i % 3 == 0:
        print(i)
```

**挑戦問題（`continue` を使う版）**
```python
for i in range(1, 21):
    # 3の倍数でなければ、以降の処理(print)をスキップして次のループへ
    if i % 3 != 0:
        continue
    print(i)
```
