# 練習問題：クラス

このファイルでは、「7.クラス.md」で学んだクラスの定義、オブジェクトの生成、継承について、いくつかの問題に挑戦してみましょう。

---

### 問題1：`Book` クラスの作成

1.  本の情報を扱う `Book` というクラスを定義してください。
2.  `__init__` メソッドを使って、オブジェクトが生成されるときに `title`（タイトル）と `author`（著者）の2つの属性を受け取り、設定できるようにしてください。
3.  `Book` クラスから、あなたの好きな本の情報を持つオブジェクトを1つ生成し、`my_book` という変数に代入してください。
4.  `my_book` の `title` と `author` をそれぞれ表示してください。

---

### 問題2：メソッドの追加

問題1で作成した `Book` クラスに、`info` というメソッドを追加してください。

1.  `info` メソッドは、`"タイトル: ○○, 著者: △△"` という形式の文字列を `return` で返すようにします。
2.  問題1で生成した `my_book` オブジェクトの `info` メソッドを呼び出し、返された結果を `print()` で表示してください。

---

### 問題3：クラスの継承

問題2で作成した `Book` クラスを継承して、`Ebook`（電子書籍）という新しいクラスを作成してください。

1.  `Ebook` クラスの `__init__` メソッドでは、`title`, `author` に加えて、`file_size`（ファイルサイズ, MB単位）も受け取るようにします。
2.  `Ebook` の `__init__` メソッド内では、`super()` を使って親クラスである `Book` の `__init__` メソッドを呼び出し、`title` と `author` の設定を任せてください。
3.  `Ebook` クラスで、`Book` クラスの `info` メソッドを**オーバーライド**してください。新しい `info` メソッドは、`"タイトル: ○○, 著者: △△, ファイルサイズ: □□MB"` という形式の文字列を返すようにします。（ヒント：`super().info()` を使うと、親クラスのメソッドの結果を再利用できます）
4.  `Ebook` クラスからオブジェクトを1つ生成し、その `info` メソッドを呼び出して結果を表示してください。


---

## 解答例

```python
# 問題1 & 2
class Book:
    # 1. __init__メソッド
    def __init__(self, title, author):
        self.title = title
        self.author = author

    # 2. infoメソッド
    def info(self):
        return f"タイトル: {self.title}, 著者: {self.author}"

# 3. オブジェクトの生成
my_book = Book("Python入門", "山田太郎")

# 4. 属性の表示
print(my_book.title)
print(my_book.author)

# 問題2の実行
print(my_book.info())

print("-----")

# 問題3
class Ebook(Book):
    # 1 & 2. __init__メソッドのオーバーライド
    def __init__(self, title, author, file_size):
        super().__init__(title, author)
        self.file_size = file_size

    # 3. infoメソッドのオーバーライド
    def info(self):
        base_info = super().info()
        return f"{base_info}, ファイルサイズ: {self.file_size}MB"

# 4. Ebookオブジェクトの生成とメソッド呼び出し
my_ebook = Ebook("独習Python", "鈴木一郎", 25)
print(my_ebook.info())

```
