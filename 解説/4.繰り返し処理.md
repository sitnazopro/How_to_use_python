# Pythonの基本：繰り返し処理 (ループ)

プログラミングにおける「繰り返し処理（ループ）」とは、同じ、あるいは似たような処理を何度も自動的に実行させるための仕組みです。Pythonでは主に `for` ループと `while` ループの2種類が使われます。

---

## 1. forループ (決まった回数繰り返す)

`for` ループは、リストやタプル、文字列などの「イテラブル（反復可能オブジェクト）」の要素を一つずつ取り出して、すべての要素に対して処理を行いたい場合に使います。

**構文:**
```python
for 変数 in イテラブル:
    # 各要素に対して実行される処理
    # この部分はインデントする
```

-   `イテラブル` から要素が1つずつ取り出され、`変数` に代入されます。
-   イテラブルの全要素に対して処理が終わると、ループは自動的に終了します。

**例1: リストの要素を順番に出力**
```python
fruits = ["apple", "banana", "cherry"]

for fruit in fruits:
    print(fruit)

# 出力:
# apple
# banana
# cherry
```

### `range()` 関数と組み合わせる

決まった回数だけ処理を繰り返したい場合は `range()` 関数が便利です。

-   `range(n)`: 0 から n-1 までの連続した数値を生成します。

**例2: 5回 "Hello" と出力**
```python
for i in range(5):  # iには 0, 1, 2, 3, 4 が順番に入る
    print(f"{i+1}回目のHello")

# 出力:
# 1回目のHello
# 2回目のHello
# 3回目のHello
# 4回目のHello
# 5回目のHello
```

## 2. whileループ (条件が満たされている間繰り返す)

`while` ループは、指定した条件式が `True` である限り、処理を繰り返し実行します。`for` ループと違い、繰り返す回数が事前に決まっていない場合に便利です。

**構文:**
```python
while 条件式:
    # 条件式がTrueのときに実行される処理
    # ループを終了させるための処理も忘れずに書く
```

-   ループの各回の開始時に `条件式` が評価されます。
-   `True` であればブロック内の処理が実行され、`False` になるとループは終了します。

**注意:** 条件式が永遠に `False` にならない場合、**無限ループ**に陥ってプログラムが停止しなくなってしまうため、ループ内で条件を変える処理（カウンターを増やすなど）を忘れないようにする必要があります。

**例: カウンターが5になるまで繰り返す**
```python
count = 0

while count < 5:
    print(f"現在のカウント: {count}")
    count = count + 1  # countに1を足して、ループの終了条件に近づける

print("ループが終了しました。")

# 出力:
# 現在のカウント: 0
# 現在のカウント: 1
# 現在のカウント: 2
# 現在のカウント: 3
# 現在のカウント: 4
# ループが終了しました。
```

## 3. ループの制御

ループの途中で処理の流れを変えたい場合があります。

### `break` (ループを中断する)

`break` が実行されると、その時点でループは即座に終了します。

**例: 特定の要素を見つけたらループを抜ける**
```python
numbers = [1, 5, 8, -3, 10]

for num in numbers:
    if num < 0:
        print("負の数を見つけたのでループを終了します。")
        break
    print(num)

# 出力:
# 1
# 5
# 8
# 負の数を見つけたのでループを終了します。
```

### `continue` (今回の処理をスキップして次に進む)

`continue` が実行されると、その回の残りの処理はスキップされ、次のループ処理（次の要素や次の条件判定）に移ります。

**例: 奇数だけを出力する**
```python
for i in range(10):  # 0から9までの数
    if i % 2 == 0:  # 偶数かどうかをチェック
        continue      # 偶数なら、この回のprintは実行せずに次に進む
    print(i)

# 出力:
# 1
# 3
# 5
# 7
# 9
```
