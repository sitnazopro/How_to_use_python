# Pythonの基本：関数

「関数」とは、特定の処理をひとまとめにしたものです。同じ処理を何度も書く必要がなくなり、プログラムが整理され、再利用しやすくなります。

Pythonには、`print()` や `len()` のようにあらかじめ用意されている**組み込み関数**と、自分で作成する**自作関数**があります。

---

## 1. 関数の定義と呼び出し

関数は `def` キーワードを使って定義します。

**構文:**
```python
def 関数名(引数1, 引数2, ...):
    # 実行したい処理
    # この部分はインデントする
    return 戻り値  # 処理の結果を返す (省略可能)
```

-   **関数名:** 処理の内容がわかるような名前を付けます。
-   **引数 (ひきすう):** 関数に渡す値です。関数内でこの値を使って処理を行います。不要な場合は `()` の中を空にします。
-   **戻り値 (もどりち):** 関数の処理結果として返される値です。`return` キーワードを使って指定します。戻り値が不要な場合は `return` を省略できます（その場合、`None` という特殊な値が返ります）。

定義した関数を使うことを「呼び出し」と言います。

**例1: シンプルな関数**
```python
# "Hello" と表示する関数を定義
def say_hello():
    print("Hello, Python!")

# 関数を呼び出し
say_hello()  # => Hello, Python!
```

---

## 2. 引数 (ひきすう)

引数を使うと、関数に情報を渡して、その情報に応じた処理を行わせることができます。

**例2: 名前を受け取って挨拶する関数**
```python
def greet(name):
    print(f"こんにちは、{name}さん！")

# 関数を呼び出すときに引数を渡す
greet("佐藤")  # => こんにちは、佐藤さん！
greet("鈴木")  # => こんにちは、鈴木さん！
```

### デフォルト引数

引数にデフォルトの値（初期値）を設定することもできます。呼び出し時にその引数が省略された場合、デフォルト値が使われます。

```python
def greet(name, message="元気ですか？"):
    print(f"こんにちは、{name}さん。{message}")

greet("田中")              # message引数を省略
# => こんにちは、田中さん。元気ですか？

greet("山田", "お疲れ様です。") # message引数を指定
# => こんにちは、山田さん。お疲れ様です。
```

---

## 3. 戻り値 (もどりち)

`return` を使うと、関数の処理結果を呼び出し元に返すことができます。返された値は、変数に代入したり、他の処理に使ったりできます。

**例3: 2つの数値を受け取って合計を返す関数**
```python
def add(a, b):
    total = a + b
    return total

# 関数を呼び出して、戻り値を変数 result に代入
result = add(5, 3)
print(result)  # => 8

# 戻り値を直接使うこともできる
print(add(10, 20)) # => 30
```

戻り値は複数指定することも可能です。その場合、タプルとして返されます。

```python
def get_point():
    x = 10
    y = 20
    return x, y  # xとyを返す

point = get_point()
print(point)      # => (10, 20)

x_coord, y_coord = get_point() # アンパックして受け取る
print(x_coord)    # => 10
print(y_coord)    # => 20
```

---

## なぜ関数を使うのか？

-   **再利用性:** 同じコードを何度も書く手間が省けます。
-   **可読性:** 処理の塊に名前を付けることで、プログラムが何をしているのか分かりやすくなります。
-   **保守性:** 修正が必要な場合、その関数の中だけを直せばよいため、メンテナンスが楽になります。

関数をうまく使うことは、分かりやすく効率的なプログラムを書くための重要なスキルです。
