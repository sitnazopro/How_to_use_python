# Pythonの基本：条件分岐 (if文)

プログラミングにおける「条件分岐」とは、「もし○○ならば、この処理をする。そうでなければ、別の処理をする」というように、条件によって実行する処理を変えることです。Pythonでは `if` 文を使ってこれを実現します。

---

## 1. 基本的なif文

最もシンプルな形は、ある条件が `True` の場合にのみ、特定の処理を実行するものです。

**構文:**
```python
if 条件式:
    # 条件式がTrueのときに実行される処理
    # この部分はインデント(字下げ)する
```

-   `条件式` には、結果が `True` または `False` になる式（比較演算子や論理演算子を使った式など）を書きます。
-   条件式の後にはコロン `:` が必要です。
-   実行する処理は、`if` 文の行よりも一段深くインデント（通常は半角スペース4つ）します。このインデントが、処理の範囲を示します。

**例:**
```python
age = 20

if age >= 18:
    print("成人です。")

# 条件がFalseなので、このブロックは実行されない
temperature = 25
if temperature > 30:
    print("エアコンをつけましょう。")
```

---

## 2. if-else文 (そうでなければ)

条件が `True` のときの処理と、`False` のときの処理（そうでなければ）を書きたい場合は `else` を使います。

**構文:**
```python
if 条件式:
    # 条件式がTrueのときに実行される処理
else:
    # 条件式がFalseのときに実行される処理
```

**例:**
```python
age = 16

if age >= 18:
    print("成人です。")
else:
    print("未成年です。")
# => 未成年です。
```

---

## 3. if-elif-else文 (さらに他の条件も)

3つ以上の条件で分岐させたい場合は、`elif` (else if の略) を使います。`elif` は複数追加することができます。

**構文:**
```python
if 条件式1:
    # 条件式1がTrueのときに実行される処理
elif 条件式2:
    # 条件式1がFalseで、条件式2がTrueのときに実行される処理
elif 条件式3:
    # ...
else:
    # すべての条件式がFalseのときに実行される処理
```

-   上から順番に条件がチェックされ、最初に `True` になったブロックの処理だけが実行されます。
-   `else` は省略可能です。

**例:**
```python
score = 85

if score >= 90:
    print("評価はAです。")
elif score >= 80:
    print("評価はBです。")
elif score >= 70:
    print("評価はCです。")
else:
    print("評価はDです。再挑戦しましょう。")

# => 評価はBです。
```

---

## 補足

`if` 文の条件式には、`bool` 値 (`True`/`False`) だけでなく、数値や文字列、リストなども指定できます。その場合、Pythonは以下のようなルールで `True`/`False` を判断します（これを **Truthiness** と言います）。

-   **`False` とみなされる値:**
    -   `False`
    -   数値の `0` や `0.0`
    -   空の文字列 `""`
    -   空のリスト `[]`、タプル `()`、辞書 `{}`
    -   `None`
-   **`True` とみなされる値:**
    -   上記以外のほとんどの値

**例:**
```python
my_list = [1, 2, 3]

if my_list:  # リストが空ではないのでTrueと判断される
    print("リストには要素があります。")
else:
    print("リストは空です。")

# => リストには要素があります。
```
