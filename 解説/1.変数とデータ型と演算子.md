# Pythonの基本：変数、データ型、演算子

## 1. 変数
変数とは、データ（値）に名前を付けて保存するための箱のようなものです。Pythonでは、 `=` を使って変数に値を代入します。

**例:**
```python
# messageという変数に "Hello, Python!" という文字列を代入
message = "Hello, Python!"
print(message)  # => Hello, Python!

# numberという変数に 100 という数値を代入
number = 100
print(number)  # => 100
```

## 2. データ型
Pythonには様々な種類のデータを扱うための「データ型」があります。ここでは基本的なものを紹介します。

| データ型 | 説明 | 例 |
| :--- | :--- | :--- |
| **整数型 (int)** | 小数点のない数値 | `10`, `-5`, `0` |
| **浮動小数点型 (float)** | 小数点のある数値 | `3.14`, `-0.01` |
| **文字列型 (str)** | 文字の集まり。シングルクォート(`'`)またはダブルクォート(`"`)で囲む | `'Hello'`, `"Python"` |
| **真偽値型 (bool)** | `True` (真) か `False` (偽) のどちらかを表す | `True`, `False` |

Pythonは**動的型付け言語**なので、変数の型を明示的に宣言する必要はありません。代入された値によって自動的に型が決まります。

**例:**
```python
x = 10         # xは整数型 (int)
y = 3.14       # yは浮動小数点型 (float)
name = "Taro"  # nameは文字列型 (str)
is_active = True # is_activeは真偽値型 (bool)
```

`type()` 関数を使うと、変数のデータ型を調べることができます。
```python
print(type(x))  # => <class 'int'>
```

## 3. 演算子
演算子とは、計算や比較などを行うための記号です。

### 算術演算子
数値の計算に使います。

| 演算子 | 意味 | 例 (`a=10, b=3`) | 結果 |
| :--- | :--- | :--- | :--- |
| `+` | 足し算 | `a + b` | `13` |
| `-` | 引き算 | `a - b` | `7` |
| `*` | 掛け算 | `a * b` | `30` |
| `/` | 割り算 | `a / b` | `3.33...` |
| `//` | 割り算（小数点以下切り捨て） | `a // b` | `3` |
| `%` | 余り | `a % b` | `1` |
| `**` | べき乗 | `a ** b` | `1000` |

### 文字列の結合とフォーマット
文字列同士を `+` で結合したり、文字列の中に変数の値を埋め込んだりする方法も見てみましょう。

**`+` を使った結合:**
```python
name = "スズキ"
age = 25
greeting = "こんにちは、" + name + "さん。あなたは" + str(age) + "歳です。"
print(greeting)
# => こんにちは、スズキさん。あなたは25歳です。
```
この方法では、`age` のような数値型の変数を `str()` を使って明示的に文字列に変換する必要があります。

**f-string (フォーマット済み文字列リテラル)**
Python 3.6から導入された、よりモダンで直感的な方法です。文字列の前に `f` を付け、埋め込みたい変数を `{}` で囲みます。

**例:**
```python
name = "サトウ"
age = 30
greeting = f"こんにちは、{name}さん。あなたは{age}歳です。"
print(greeting)
# => こんにちは、サトウさん。あなたは30歳です。
```
f-string を使うと、`str()` で変換する必要がなく、コードが非常にスッキリして読みやすくなります。特別な理由がない限り、文字列のフォーマットには f-string を使うのがおすすめです。

### 比較演算子
2つの値を比較し、結果を `True` または `False` で返します。

| 演算子 | 意味 | 例 (`a=10, b=5`) | 結果 |
| :--- | :--- | :--- | :--- |
| `==` | 等しい | `a == 10` | `True` |
| `!=` | 等しくない | `a != b` | `True` |
| `>` | より大きい | `a > b` | `True` |
| `<` | より小さい | `a < b` | `False` |
| `>=` | 以上 | `a >= 10` | `True` |
| `<=` | 以下 | `a <= b` | `False` |

### 論理演算子
複数の条件を組み合わせるために使います。

| 演算子 | 意味 | 例 |
| :--- | :--- | :--- |
| `and` | **かつ** (両方の条件が`True`なら`True`) | `x > 0 and y > 0` |
| `or` | **または** (どちらかの条件が`True`なら`True`) | `age < 20 or age >= 65` |
| `not` | **ではない** (`True`と`False`を逆転させる) | `not is_empty` |

これらの基本をマスターすることが、Pythonプログラミングの第一歩です。
