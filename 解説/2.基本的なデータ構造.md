# Pythonの基本：基本的なデータ構造

Pythonには、複数のデータをまとめて扱うための便利な「データ構造」が組み込まれています。ここでは、特に重要な4つのデータ構造を紹介します。

1.  **リスト (List)**
2.  **タプル (Tuple)**
3.  **辞書 (Dictionary)**
4.  **セット (Set)**

---

## 1. リスト (List)

リストは、複数のデータを順序付けて格納できる、最も一般的なデータ構造です。角括弧 `[]` を使って作成し、カンマ `,` で各要素を区切ります。

-   **順序がある** (インデックスで要素を指定できる)
-   **変更可能** (要素の追加、変更、削除ができる)
-   異なるデータ型を混在させることができる

**例:**
```python
# フルーツのリストを作成
fruits = ["apple", "banana", "cherry"]
print(fruits)  # => ['apple', 'banana', 'cherry']

# インデックスを使って要素にアクセス (0から始まる)
print(fruits[0])  # => apple

# 要素を変更
fruits[1] = "orange"
print(fruits)  # => ['apple', 'orange', 'cherry']

# 要素を追加
fruits.append("grape")
print(fruits)  # => ['apple', 'orange', 'cherry', 'grape']

# 要素を削除
fruits.remove("cherry")
print(fruits)  # => ['apple', 'orange', 'grape']
```

---

## 2. タプル (Tuple)

タプルはリストと似ていますが、一度作成すると**変更できない** (immutable) という大きな違いがあります。丸括弧 `()` を使って作成します。

-   **順序がある**
-   **変更不可能** (定数的なデータの集まりに使う)

**例:**
```python
# 座標を表すタプル
point = (10, 20)
print(point)  # => (10, 20)

# インデックスでアクセス
print(point[0])  # => 10

# タプルの要素は変更できない (エラーになる)
# point[0] = 15  # => TypeError: 'tuple' object does not support item assignment
```

変更されたくないデータを扱う場合に便利です。

---

## 3. 辞書 (Dictionary)

辞書は、**キー (Key)** と **値 (Value)** のペアでデータを格納するデータ構造です。波括弧 `{}` を使って作成します。他のデータ構造と違い、順序ではなくキーを使ってデータにアクセスします。

-   キーと値のペアで管理
-   キーは一意 (重複不可) でなければならない
-   **変更可能**

**例:**
```python
# 人の情報を格納する辞書
person = {
    "name": "Taro",
    "age": 25,
    "city": "Tokyo"
}
print(person)  # => {'name': 'Taro', 'age': 25, 'city': 'Tokyo'}

# キーを使って値にアクセス
print(person["name"])  # => Taro

# 値を変更
person["age"] = 26
print(person)  # => {'name': 'Taro', 'age': 26, 'city': 'Tokyo'}

# 新しいキーと値のペアを追加
person["job"] = "Engineer"
print(person)  # => {'name': 'Taro', 'age': 26, 'city': 'Tokyo', 'job': 'Engineer'}
```

---

## 4. セット (Set)

セットは、**重複しない**要素の集まりを格納するデータ構造です。順序はありません。波括弧 `{}` または `set()` 関数を使って作成します。

-   **順序がない**
-   **要素の重複を許さない**
-   和集合、積集合などの集合演算ができる

**例:**
```python
# セットを作成 (重複した"apple"は1つになる)
fruits = {"apple", "banana", "cherry", "apple"}
print(fruits)  # => {'banana', 'cherry', 'apple'} (順序は保証されない)

# 要素がセットに含まれているか確認
print("banana" in fruits)  # => True
print("grape" in fruits)   # => False

# セットの演算
a = {1, 2, 3}
b = {3, 4, 5}

# 和集合 (aとbの両方に含まれる要素)
print(a | b)  # => {1, 2, 3, 4, 5}

# 積集合 (aとbの共通要素)
print(a & b)  # => {3}
```

リストから重複する値を取り除きたい場合などによく使われます。

```python
numbers = [1, 2, 2, 3, 4, 3, 5]
duplicates_removed = list(set(numbers))
print(duplicates_removed)  # => [1, 2, 3, 4, 5] (順序は元に戻らない可能性がある)
```
